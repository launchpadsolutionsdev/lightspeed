Data Privacy

Your organization’s data is never used to train AI models — per Anthropic’s API terms, data sent through the Claude API is not used for model training. Each organization’s knowledge base is isolated at the application level, and all traffic between your browser and our servers is encrypted via HTTPS.

Infrastructure Security

AI processing runs through Anthropic’s SOC 2 compliant infrastructure. All data in transit is encrypted over HTTPS. We use role-based access controls to restrict who can access what, and every sensitive action is recorded in our audit log.

Access Control

Authentication through Google and Microsoft SSO with industry-standard OAuth 2.0. Role-based permissions (owner, admin, member) let you control who can manage settings, invite users, and access tools. All access is logged and auditable.

Our security commitments

✓

No model training on your data

Under Anthropic’s API terms, data sent through the Claude API is not used for model training. We do not train our own models on your content.

✓

Tenant isolation

Each organization’s data is logically isolated at the application level. All database queries are scoped to your organization — no cross-tenant access is possible through the application.

✓

Data export

Organization admins can export all of their organization’s data at any time. For account deletion requests, contact our team directly.

✓

Comprehensive audit logging

Every sensitive action — member invitations, role changes, settings updates, data exports — is recorded with timestamps, user IDs, and IP addresses.

✓

No third-party data sharing

We never sell or share your data with third parties. The only external services that process your data are Anthropic (AI) and Stripe (billing).

✓

Incident transparency

In the event of a security incident, we commit to prompt disclosure and clear communication with affected customers.
