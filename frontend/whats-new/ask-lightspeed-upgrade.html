Ask Lightspeed Gets Six Powerful New Capabilities

Ask Lightspeed has evolved from a simple Q&A tool into a full-featured knowledge assistant with persistent conversations, source citations, teach mode, and team collaboration.

Ask Lightspeed was built to let lottery operators ask questions and get instant, knowledge-base-grounded answers. But as teams started relying on it for more complex workflows, the limitations of a single question-and-answer format became clear. Teams needed to dig deeper, refine answers, and share findings with colleagues.

Today's upgrade transforms Ask Lightspeed into the kind of tool that can handle those workflows. Six major new features work together to make Ask Lightspeed a true knowledge assistant for your organization.

What's new

  • Persistent conversations — Your conversation history is now saved automatically. Pick up where you left off, revisit past questions, and build on previous answers without starting over
  • Follow-up refinements — Ask follow-up questions within the same conversation thread. Each response builds on the context of the entire conversation, producing more relevant answers the deeper you go
  • Source citations — Every response now includes citations linking back to the specific knowledge base entries used. Click any citation to see the source material and verify accuracy
  • Teach mode — When Lightspeed doesn't have the answer, teach mode lets you add the correct information directly to your knowledge base from within the conversation — no need to switch to the KB editor
  • Context management — Control which knowledge base categories and documents are included in each conversation. Focus the AI on the most relevant information for the question at hand
  • Team collaboration — Share conversations with teammates, bookmark useful threads, and build a searchable library of institutional knowledge that the whole team can access

How it works

The Ask Lightspeed interface now features a conversation sidebar that lists your recent and bookmarked threads. Each conversation maintains full context, so follow-up questions understand what came before. Source citations appear inline with a clickable reference number, and teach mode activates with a single click when you want to correct or expand the knowledge base.

This upgrade also includes an engine upgrade to Claude Sonnet 4.6, which powers more accurate, nuanced responses across all six new capabilities.

What's next

We're working on conversation templates, scheduled reports, and deeper integration between Ask Lightspeed and the Response Assistant so insights from one tool flow seamlessly into the other.
